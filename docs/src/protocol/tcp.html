TCP/IP 协议族

TCP/IP 协议族

不同计算机运行的不同协议
计算机 协议
路由器 ARP，IP，ICMP，UDP，SNMP，RIP。
客户端 ARP，IP，ICMP，UDP，TCP，DNS，HTTP，FTP。
电脑 TELNET，SMTP，POP3，DHCP，SSH
固件 ARP，IP，ICMP，UDP，BOOTP，TFTP
因特网协议栈中的层

将 TCP / IP 参考模型映射到 OSI 模型。

OSI 模型 TCP / IP 系列协议
应用层 HTTP、SMTP、FTP、Telnet、SSH、NFS、
表示层 XDR、ASN.1、SMB、AFP、NCP
会话层 ASAP、RPC、NetBIOS、ASP、Winsock、BSD sockets
传输层 TCP、UDP、TLS、SCTP、SPX、ATP、IL
网络层 IP、ARP、X.25、L2TP、PPPoE、 STP
数据链路层 以太网、IEEE 802.11、FDDI、PPPoE、L2TP、
物理层 线路、无线电、光纤

通常人们认为 OSI 模型的最上面三层在 TCP/IP 是一个层。

OSI 模型 TCP / IP 系列协议
[应用层][2] HTTP、FTP、DNS
[传输层][2] TCP、UDP、RTP、SCTP
[网络互连层][3] 对于TCP/IP来说这是因特网协议（IP）
[网络接口层][4] 例如以太网、Wi-Fi、MPLS等。
